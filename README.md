# 🛒 CRUD University Project

A simple terminal-based CRUD system developed as a university project.
Built with Python, using hash maps (dictionaries) for efficient O(1) data access instead of lists.

## Features
- List products
- Add products
- Check stock
- Update product name, price, and stock status
- 3 pre-registered products with randomized stock status on each run
- Basic error handling for invalid inputs
